# Digital Desktop

A high-fidelity Windows 11 desktop experience built with Next.js, Tailwind CSS, and Framer Motion.

## Features

- **Windows 11 UI**: Authentic desktop grid, taskbar, start menu, and window management.
- **Interactive Desktop**: Drag-and-drop icons, selection box, and context menus.
- **Window System**: Minimize, maximize, close, and window layering (z-index).
- **Claude AI App**: A built-in chat application simulating the Claude interface.

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS
- **State Management**: Zustand
- **Drag & Drop**: @dnd-kit/core
- **Icons**: Lucide React & Windows 11 Assets

## Getting Started

1. Install dependencies:
   ```bash
   yarn install
   ```

2. Run the development server:
   ```bash
   yarn dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser.
